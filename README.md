# Entrada-analogica--esp32
Voltímetro VU IoT com ESP32 para leitura de entrada analógica via ADC, exibindo níveis de tensão em tempo real por LEDs (baixo, médio e alto). Projeto embarcado com potencial para expansão IoT, permitindo monitoramento remoto via WiFi e integração com sistemas em nuvem.
